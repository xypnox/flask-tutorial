# flask-tutorial
The flask turtorial from official docs



## Steps

Install `python3-venv`

Then do :

```bash
$ python3 -m venv venv
```

To change to the virtual environment:

```bash
$ . venv/bin/activate
```

Now install all the requirements:

```bash
$ pip install flask
```



Now to run a flask app in app.py:

```bash
$ export FLASK_APP=app.py
```

```
$ flask run
```
